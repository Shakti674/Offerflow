# System Architecture

## Architecture Style

OfferFlow follows a layered modular architecture.

## Layers

- Presentation Layer
- Application Layer
- Domain Layer
- Persistence Layer
- Infrastructure Layer

## Modules

- Authentication
- User Management
- Offer Management
- Rule Engine
- Notification
- Audit

## External Systems

- PostgreSQL
- Redis
- Apache Kafka
