# OfferFlow

Enterprise Offer Management Platform built using Java 21 and Spring Boot.

---

## Overview

OfferFlow is a production-inspired backend application that demonstrates enterprise Java development using modern software engineering practices.

The platform allows organizations to create, manage, approve and publish configurable business offers through secure REST APIs.

This project is being developed incrementally following real-world engineering practices rather than as a tutorial application.

---

## Technology Stack

- Java 21
- Spring Boot 4
- PostgreSQL
- Spring Data JPA
- Spring Security (JWT)
- Redis
- Apache Kafka
- Docker
- Maven
- GitHub Actions
